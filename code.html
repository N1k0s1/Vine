FULL YSWS GUIDE

START HERE

Use this exact flow to go from zero to shipped vinyl. Learn Sonic Pi, compose your track, package your repo, and submit with confidence.

Guide 01

Install + Learn Sonic Pi

  • 1Install Sonic Pi from the official download page for macOS, Windows, or Linux.
  • 2Work through the built-in tutorial panel and run the code as you read.
  • 3Practice live_loop, sample, use_synth, and sleep until you can build a short groove.
  • 4Save your first playable version as main.rb so you have a baseline track.

Sonic Pi Sites

sonic-pi.net (download) Official Sonic Pi tutorial Mehackit quickstart lessons Sonic Pi community forum

Guide 02

Compose + Export Audio

  • 1Build a full 60-120 second track with structure: intro, core loop, variation, ending.
  • 2Use comments to label each section so reviewers can follow your arrangement choices.
  • 3Record clean audio from Sonic Pi and export to .wav or high-quality .mp3.
  • 4Listen once on headphones and once on speakers. Fix clipping, balance levels, and re-export.

# tiny starter pattern

> use_bpm 100

> live_loop :kick do

> sample :bd_haus, amp: 1.7

> sleep 0.5

> end

Guide 03

Package + Submit

  • 1Create a public GitHub repository with your final Sonic Pi code and exported audio.
  • 2Add a README that explains inspiration, technique, and how to run your track locally.
  • 3Use the Vine example submission layout so your repo is easy to review quickly.
  • 4Submit the form with your repo link and audio, then wait for pressing and shipping updates.

QUESTIONS?

Does it cost anything? expand_more
Nope. Zip. Zero. Hack Club is footing the bill for the records, the pressing, and the shipping. You just provide the code & the idea.
Who can join? expand_more
Any teenager (13-18) worldwide. If you can code it, we'll try to ship it.
What is Sonic Pi? expand_more
It's a code-based music creation and performance tool. It uses Ruby and it's incredibly powerful for algorithmic composition.
How long do I have? expand_more
Vine is a limited-run YSWS. We have a set number of records to press. Once we hit our limit, we close submissions. Ship fast!
Can I work in a team? expand_more
Submit individually by default. If team entries open officially, Vine will publish updated rules.
Can I double-dip with other Hack Club events? expand_more
Usually no. Keep your Vine submission to this event unless both programs explicitly allow overlap.
What exactly is required in my submission? expand_more
Include a public GitHub repo with your Sonic Pi code, an audio export of your track, and a README that explains your song idea!
